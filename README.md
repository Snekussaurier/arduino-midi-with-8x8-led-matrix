# How to build an Arduino midi with 8x8 LED Matrix

**Parts list:**
-   1x Arduino Mega 2560
-   1x Arduino Uno
-   3x Tlc 5940N
-   72x Common Anode RGB LED's
-   68x Pushbuttons
-   Wires
-   1x Switch potentiometer (B50K)
-   1x 6mm potentiometer knob
-   Silverwire
-   1x Acrylglas
-   1x Adafruit Music Maker Shield
